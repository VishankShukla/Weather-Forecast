# 🌤️ Weather App

A simple and responsive weather application that allows users to check real-time weather information for any city.

---

## 🚀 Features

* 🌍 Search weather by city name
* 🌡️ Displays temperature in Celsius
* ☁️ Shows weather description
* ❌ Error handling for invalid cities
* ⚡ Fast and lightweight UI

---

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
* Weather API from OpenWeather

---

## 📦 Installation & Setup

1. Clone the repository:

```bash
git clone https://github.com/your-username/weather-app.git
```

2. Navigate to project folder:

```bash
cd weather-app
```

3. Open `index.html` in your browser

---

## 🔑 API Setup

This project uses an API key from OpenWeather.

1. Go to OpenWeather website
2. Create an account
3. Generate an API key
4. Replace the API key in `script.js`:

```js
const API_Key = "YOUR_API_KEY";
```
